# Contest Alert 
```text
Contest Alert is a chrome extension which can be used for getting upcoming competitive programming contest alert and  
list. It is an awesome thing for a programmer and it is a time saver extension.  
Features : 
1. Upcoming Contest Lists
2. Remaining Time of the Contests
3. Notification before the contests
```

```text
Installing procedures in local PC:
1. Download this repo from GitHub and extract the zip file.
2. Open Google Chrome and Go to Extension Menu.
3. Enable Developer Mode from the right side.
4. Click on Load unpacked and upload the extracted folder of the extension.
5. Now, you can use it from Extensions Menu
```
